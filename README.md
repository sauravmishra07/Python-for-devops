# Python for DevOps

This repository contains Python scripts and examples for DevOps-related tasks. It includes basic Python programming concepts and data structures implementations.

## Folder Structure

- **basics/**: Contains fundamental Python scripts
  - `hello.py`: A simple "Hello World" program
  - `function.py`: Examples of Python functions
  - `check_cpu.py`: Script to check CPU usage
  - `check_env.py`: Script to check environment variables
  - `tempCodeRunnerFile.py`: Temporary file for code execution

- **data_structures/**: Contains data structure implementations
  - `list.py`: Examples and operations on Python lists

## Requirements

- Python 3.x

## Usage

Navigate to the desired folder and run the scripts using Python:

```bash
python basics/hello.py
python data_structures/list.py
```
